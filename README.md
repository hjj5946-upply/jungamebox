## 🚀 프로젝트 소개

**J GameBox**는 브라우저만 있으면 누구나 즐길 수 있는  
간단한 미니게임 컬렉션입니다.  
별도의 설치 없이 즉시 실행되며, 모바일·데스크탑 모두 대응합니다.

> ⚡ 빠른 반응속도 / 🎨 심플한 UI / 🧋 무료 & 오픈 액세스

🔗 **사이트 바로가기:** [https://jun-gamebox.com](https://jun-gamebox.com)

---

## 🧩 주요 기능

### 🎯 인기 게임

| 게임 | 설명 |
|------|------|
| 🧋 **음료내기** | 친구들과 함께하는 재밌는 음료 내기 게임 (1빠정하기) |
| 🍀 **로또번호생성** | 무작위 로또 번호 자동 생성기 |
| ⚖️ **밸런스월드컵** | 고민되는 선택지, 밸런스 게임으로 결정하기 |
| 🎭 **라이어게임** | 재밌는 심리 게임, 라이어를 찾아라! |
| 🎲 **제비뽑기** | 공정한 제비뽑기로 순서 정하기 |
| ⚔️ **무기강화하기** | 도박의 재미! 무기를 강화해보세요 |
| 🎰 **슬롯머신** | 클래식한 슬롯머신 게임 |
| 🎯 **돌려돌림판** | 룰렛으로 팀 나누기, 순서 정하기 |

### 🎮 기타 게임

| 게임 | 설명 |
|------|------|
| 🎲 **주사위** | 주사위 굴리기, 숫자 맞추기 미니게임 |
| ✂️ **가위바위보** | 기본 중의 기본, 단판 승부용 게임 |
| 🪩 **핀볼게임** | 가볍게 즐기는 실험적 미니게임 |
| 🧠 **반사신경** | 반사신경 테스트 게임 |
| ⏱️ **타이밍캐치** | 정확한 타이밍을 잡아보세요 |
| 🎯 **만약에** | 재밌는 가정 질문 게임 |
